CHAT APPLICATION - GUIDE D'UTILISATION

1. DÉMARRAGE DU SERVEUR
   - Lancer ServerLauncher.java
   - Le serveur écoute sur les ports 12345 (TCP) et 12346 (UDP)

2. DÉMARRAGE DES CLIENTS
   - Lancer ClientLauncher.java (console) OU ClientGUI.java (interface)
   - Entrer un nom d'utilisateur
   - Commencer à chatter!

3. COMMANDES DISPONIBLES
   /help     - Afficher l'aide
   /list     - Lister les utilisateurs
   /udp msg  - Envoyer via UDP
   /quit     - Quitter

4. TÉLÉCHARGEMENT
   - Exporter: File → Export → Java → Runnable JAR file
   - Choisir "Package required libraries"
   - Sélectionner l'application principale
